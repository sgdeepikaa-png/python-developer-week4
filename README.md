# python-developer-week4
Q1. When would you choose a set over a list?

I would choose a set when duplicate values need to be removed. In the playlist program, a set removes repeated song names automatically.

Q2. What is a named tuple?

A named tuple is a tuple with named fields. It makes the program easier to read because we use names like student.name instead of indexes like student[0].

Q3. What is the difference between the for-loop and comprehension methods?

The for-loop method is faster because it counts words in one pass through the list. The dictionary comprehension using count() is shorter and easier to read, but it is slower for large files because it scans the list repeatedly.
